## Zipkin instrumentation for PHP
* 用于阿里云链路追踪[TracingAnalysis](https://www.aliyun.com/product/xtrace)
* 增强官方库使其更易于使用, 更友好与调用者
* 由于官方库的Composer停留在`1.3.2`版本, 所以Fork后自行生成Composer包


## Installation
`composer require aliyun-sdk/ta-zipkin-php`
